# Agent 05 — PDF Generator

## Your job
Implement `components/ReportPDF.tsx` and wire up the download button.

---

## ReportPDF.tsx
Use `@react-pdf/renderer`. Props: `scoreResult: ScoreResult`, `orgName?: string`

Page layout (A4):
1. Header — "GivingArc" in teal (#1D9E75), org name if provided
2. Title — "990 Health Score Report"
3. Score box — large score number, grade, percentile text
4. Metrics table — 6 rows: Name | Your Value | Benchmark | Grade
5. Recommendations — only for C/D/F metrics, as bullet points
6. Footer — "Generated by GivingArc · givingarc.com/schedule-a-call/"

Keep it simple. No images. Text and colored boxes only.

---

## Wire up download button on app/results/page.tsx

```tsx
import { pdf } from '@react-pdf/renderer'
import { saveAs } from 'file-saver'

// npm install file-saver @types/file-saver

async function handleDownload() {
  const blob = await pdf(
    <ReportPDF scoreResult={scoreResult} orgName={orgName} />
  ).toBlob()
  saveAs(blob, '990-health-score.pdf')
}
```

Button: "Download PDF Report" — triggers handleDownload on click.

## Done when
Clicking the button downloads a readable branded PDF.
PDF opens correctly in browser PDF viewer.
