# Agent 03 — UI

## Your job
Build all pages and components. Use Tailwind only. No inline styles.
Primary color class: use `bg-[#1D9E75]` and `text-[#1D9E75]` for teal.

## Grade color map (use throughout)
```ts
const gradeColor = {
  A: '#1D9E75', // teal
  B: '#3B82F6', // blue
  C: '#F59E0B', // amber
  D: '#F97316', // orange
  F: '#EF4444', // red
}
```

---

## 1. QuizForm.tsx
A form with 7 fields in 2 sections.

**Section A — Revenue (Part VIII)**
- Total Revenue → Part VIII, Line 12
- Program Service Revenue → Part VIII, Line 2g
- Contributions & Grants → Part VIII, Line 1h

**Section B — Expenses (Part IX)**
- Total Expenses → Part IX, Line 25
- Program Service Expenses → Part IX, Line 25, Col B
- Management & General → Part IX, Line 25, Col C
- Fundraising Expenses → Part IX, Line 25, Col D

Each field: label, Form 990 reference in gray text below, dollar-formatted input.
Validate on submit: all required, all positive, show error if programExpenses + mgmtExpenses + fundraisingExpenses > totalExpenses.

On submit: encode inputs as base64 JSON → navigate to `/results/gate?d=<encoded>`

---

## 2. ScoreCircle.tsx
Props: `score: number`, `grade: string`
SVG circle, strokeDasharray to show progress.
Large score number in center, grade letter below it.
Color matches grade (use gradeColor map).
Size: 180px × 180px.

---

## 3. MetricCard.tsx
Props: `metric: MetricResult`
Card with: metric name, your value (large), benchmark (small gray), grade badge, insight text.
If metric has recommendation: show it in a light yellow box below.

## 6. app/results/gate/page.tsx
- Read `?d=` param from URL, pass it through to /results
- Show a blurred/opacity-50 fake score card as preview
- Overlay card with:
  - "Your 990 Health Score is ready"
  - Email input (required)
  - First Name input (optional)
  - Org Name input (optional)
  - Checkbox: "Send me nonprofit accounting tips" (unchecked by default)
  - Submit button → POST to /api/send-report → then navigate to `/results?d=<same>`

---

## 7. app/results/page.tsx
- Read and decode `?d=` param → call `calculateScore()`
- If decode fails → redirect to /quiz
- Render:
  1. `<ScoreCircle />` with total score + grade
  2. Percentile label: "Better than {percentile}% of similar nonprofits"
  3. Grid of 6 `<MetricCard />` components
  4. Recommendations section (only metrics graded C/D/F)
  5. CTA: "Want help improving these numbers? Schedule a free consult →" (link to https://givingarc.com/schedule-a-call/)
  6. "Download PDF Report" button (wired in Agent 05)

## Done when
All pages render at localhost:3000 with no console errors.
Quiz form encodes data and navigates correctly.
Results page decodes and shows score.
